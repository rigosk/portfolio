Rail timetable

Timetable developed in Python using the plotly library

Modal title
In order to avoid application dependency, I developed the timetable in python, and the exported in html file. In this way, the timetable can be read interactively in a Chrome window. With the plotly library we can zoom, hide or show trains, display the capacity allocation and print in a very simple way. An example can be seen at link.

Passengers flow

The flow of passengers using python and plotly library.

Modal title
In order to avoid application dependency, I developed the passengers flow in python, and the exported in html file. In this way, the passengers flow can be read in a Chrome window. In this example the passengers flow of the night train for the year 2018. It is illustrated, that ~85% of available seats are sold and ~80% of them are booked for the final station (Salonica).
